# Security and Responsible Use

This project is designed for low-frequency notification-only monitoring.

Do not use this project to:

- bypass CAPTCHA or bot checks;
- bypass queues;
- automate login;
- automate ticket selection;
- automate checkout or payment;
- generate high traffic against ticket websites.

If a monitored website displays a CAPTCHA or bot-check page, this project should stop and notify the user to handle it manually.

Please respect website terms of service and local laws.
